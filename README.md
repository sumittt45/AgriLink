# 🌱 AgriLink

**Fair for Farmers, Fresh for You**

AgriLink is a modern B2B agricultural marketplace that connects **farmers directly with buyers** — eliminating middlemen and ensuring fair pricing for both sides.

---

## 🚀 Overview

AgriLink helps:

* 👨‍🌾 Farmers sell directly to buyers
* 🛒 Buyers purchase fresh crops at fair prices
* 🤝 Both sides negotiate transparently

Built with a focus on **real-world usability, scalability, and clean UX**.

---

## ✨ Key Features

### 👨‍🌾 Farmer Side

* Create and manage farm profile
* Add crops with pricing and quantity
* Receive and respond to buyer quotes
* Dashboard with orders, quotes, and analytics

### 🛒 Buyer Side

* Browse farmers and crops
* Place orders and negotiate prices
* View farmer details, ratings, and reviews
* Location-based recommendations

### 🤖 AI Crop Forecast (Experimental)

* Select State, City, and Language
* Get insights on:

  * 📈 High demand crops
  * 📉 Low demand crops
  * 🌾 Smart farming suggestions

## 💳 Payments

AgriLink uses Razorpay for secure online payments.

* Supports UPI, Cards, Net Banking, Wallets
* Payment is processed via Razorpay Checkout
* Backend verifies payment before confirming order
* Sensitive keys are securely stored (no exposure)

**Test Mode Enabled** – no real money is charged.

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite + Tailwind CSS
* **Backend:** Supabase (Auth, Database, Storage)
* **AI:** Google Gemini API
* **Deployment:** Vercel
